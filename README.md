# Trader Performance vs Market Sentiment Analysis

Data Science Intern Assignment for Primetrade.ai

## Project Overview
Analysis of how Bitcoin Fear & Greed sentiment affects trader behavior and performance on Hyperliquid.

## Files
- `trade_sentiment_analysis.ipynb` → Main analysis notebook
- Two datasets used (not included due to size):
  - Bitcoin Market Sentiment (Fear/Greed)
  - Historical Trader Data (Hyperliquid)

## How to Run
1. Download the two datasets from the original Google Drive links provided in the assignment.
2. Place them in the same folder as the notebook (or update the paths in the first cells).
3. Open `trade_sentiment_analysis.ipynb` in Jupyter Notebook / VS Code / Google Colab.
4. Run all cells from top to bottom.
